Get Started

To get started with using andd customisingthe components you have to Copy and paste the code in the head tag of your html document. Make sure that your custom style sheet is linked below the pasted code.

Note: You will need a FontAwesome CDN in your project to ensure the proper working of all the components. Also, the root font size is set to 13px.
